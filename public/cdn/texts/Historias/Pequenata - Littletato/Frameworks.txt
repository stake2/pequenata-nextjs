Pequenata Framework:
1. Write Pequenata on Wattpad
2. Revise on Grammarly
3. Translate into Portuguese and revise
4. Insert it into the website
_Open the PHP file
__Use Capítulo.py to make a chapter HTML-friendly text file
___Update the variable chapter counter
5. Publish it on Wattpad
6. Update the website on Netlify using the Publish.bat from Core.py
7. Send the link to my special readers (Igor, Ana, Amanda, Cah, Agnes, Giulia, Drê) on Discord
8. Share the link on Twitter and Facebook
